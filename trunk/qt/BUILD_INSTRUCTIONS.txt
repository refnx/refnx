packaging for OSX:

1) You need to change the options line to:
OPTIONS = {'argv_emulation': True, 'includes':['matplotlib.tri','matplotlib.projections.geo', 'matplotlib.projections.polar', 'matplotlib.tri.triplot','matplotlib.tri.tripcolor','matplotlib.tri.tricontour', 'matplotlib.tri.triangulation', 'matplotlib.delaunay.triangulate','sip', 'PySide', 'PySide.QtGui', 'PySide.QtCore', 'pyplatypus', 'pyplatypus.analysis', 'pyplatypus.dataset', 'pyplatypus.util']}

2) Use the following line to build the app:

python setup.py py2app --iconfile shuttle_1403756i.icns

3) The app will try to load two sets of Qt dylibs. Why this is I have no idea. To prevent this problem create the file:

motofit.app/Contents/Resources/qt.conf

It just needs to be a blank file.
